Gas Town has been added to the Open Orchestrators directory as a coding-agent workspace manager.

The project fits the directory because orchestration is the product surface: it focuses on coordinating multiple coding agents, keeping work state persistent, and making handoffs legible across tools like Claude Code, GitHub Copilot, Codex, and Gemini.

For the directory, Gas Town represents a practical branch of the category: not only agent frameworks or visual builders, but workspaces that help a person run several coding agents against real project work.
