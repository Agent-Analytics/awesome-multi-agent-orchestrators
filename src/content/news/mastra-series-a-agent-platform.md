Mastra announced on April 9, 2026 that it raised a $22M Series A led by Spark Capital, bringing total funding to $35M. The same announcement also frames Mastra as more than a TypeScript agent framework: the company is launching a platform with Studio, Server, and Memory Gateway.

This matters for Open Orchestrators readers because Mastra is moving deeper into the operational layer around agents. The announcement points to platform needs that keep coming up across the category: evals, logs, traces, datasets, deployment, memory, auth, and RBAC for real agent systems.

For builders and operators, the practical implication is that Mastra is trying to own more of the path from local agent code to production agent operations. That makes Mastra worth watching not only as a framework, but as a broader agent platform for teams that need observability, deployment, and memory around their workflows.
